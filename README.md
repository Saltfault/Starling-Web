# Starling Web

Web client for [Starling](https://forgejo.hearthhome.lol/Saltfault/Starling) —
a federated peer-to-peer communications platform.

## Status

🚧 Not yet started. Will provide text, voice, and video calls in the browser
via WebAssembly.

## Dependencies

- [starling-server](https://forgejo.hearthhome.lol/Saltfault/Starling-Server) — shared protocol library
